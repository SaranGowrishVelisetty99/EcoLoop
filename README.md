# EcoLoop Upcycling Engine

A full-stack Next.js app for scanning discarded items, generating upcycling blueprints with OpenRouter, and tracking sustainability impact in Firebase.

## Features

- Firebase Auth sign-in/sign-up
- Firebase Storage upload flow
- Firestore scan and project records
- OpenRouter AI analysis route with structured JSON enforcement
- Responsive dashboard and project blueprint pages

## Setup

1. Copy `.env.example` to `.env.local` and fill in your Firebase and OpenRouter values.
2. Run `npm install`.
3. Run `npm run dev`.
